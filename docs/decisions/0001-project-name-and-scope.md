# 0001: Project Name And Scope

## Decision

Use **Frontend Agency** as the project and product name.

Use these identifiers:

- Repository: `frontend-agency`
- Marketplace: `frontend-agency`
- Core plugin: `frontend-agency-core`

## Rationale

The name is direct, professional, memorable, and aligned with the repository's real purpose: giving AI agents the behavior of a disciplined frontend agency across strategy, design, implementation, and review.

The project will not use abstract naming that obscures the product's function.

## Scope

Frontend Agency provides modular skills and plugin packaging for AI agents. Claude Code is the first-class runtime target, but the skills are written to remain useful for any agent that understands the Agent Skills structure.
