# Research Summary

The Deep Research input identified recurring weaknesses in AI-generated frontends:

- Generic visual style without brand distinctiveness
- Weak semantic HTML and accessibility gaps
- Poor responsive behavior
- Missing SEO and performance discipline
- Shallow asset, motion, and review workflows
- Overreliance on one-shot generation instead of iterative critique

Frontend Agency turns those findings into a modular skill system:

- Strategy before implementation
- Design language before component production
- Review gates before delivery
- Accessibility, SEO, performance, and visual QA as first-class checks

Detailed source artifacts are kept locally and ignored by Git until a curated research package is prepared.
