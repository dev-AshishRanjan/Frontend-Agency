- Finalize marketplace metadata
- Add install and update documentation
- Add release checklist

## Phase 4: Agent Compatibility

- Document usage outside Claude Code
- Add adapter notes for agents that support Agent Skills
- Keep the canonical skill source format vendor-neutral
