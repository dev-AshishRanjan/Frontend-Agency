# Landing Page Strategy

Plan the narrative structure, conversion strategy, and content architecture for a marketing or landing page.

## When to Run

Run before building or reviewing any marketing page, landing page, hero section, or conversion-oriented surface.

## Workflow

1. Review brand discovery and creative direction outputs.
2. Define the primary conversion goal and the single desired user action.
3. Map the user's mental model: awareness → interest → desire → action.
4. Plan section hierarchy: what appears first, what earns the scroll.
5. Define proof strategy: what evidence closes objections at each stage.
6. Define CTA strategy: primary CTA, secondary CTA, placement, and copy direction.
7. Identify content requirements: copy, imagery, social proof, data, and demos.

## Output

Return:

- **Conversion goal** — the single metric this page optimizes for
- **Narrative arc** — section sequence with purpose of each section
- **Section briefs** — content requirements and design intent for each section
- **Proof strategy** — what evidence appears where and why
- **CTA strategy** — placement, copy direction, and fallback path
- **Content production list** — what copy, assets, and data must be produced
- **Above-the-fold requirements** — what must be visible without scrolling on all breakpoints
