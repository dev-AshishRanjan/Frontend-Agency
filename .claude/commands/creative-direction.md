# Creative Direction

Translate brand and audience context into a concrete, actionable creative direction.

## When to Run

Run before any design, visual system, or layout work begins — especially when brand tone, visual style, or creative strategy has not been explicitly defined.

## Workflow

1. Review brand discovery output or extract brand attributes if no discovery has been run.
2. Define the emotional register — what the user should feel at first glance.
3. Select visual references and anti-references (what the design must not look like).
4. Define design personality across key axes: minimal/expressive, editorial/functional, bold/refined.
5. Produce a creative brief that visual-language and design-system skills can execute directly.

## Output

Return:

- **Visual personality** — three descriptors with rationale
- **Emotional register** — the desired emotional response at each user touchpoint
- **Design references** — 3–5 reference directions with specific rationale for each
- **Anti-references** — what to avoid and why
- **Creative constraints** — mandatory brand elements, off-limits patterns, accessibility requirements
- **Success criteria** — how to evaluate whether the direction was executed correctly
